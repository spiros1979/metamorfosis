# 1ο Μέρος: Σχεδίαση Διαδραστικού AR Κολάζ (Kafka)

## Επιλεγμένο λογοτεχνικό έργο

- *Η Μεταμόρφωση* του Franz Kafka (σε απόσπασμα για σχολική αξιοποίηση).

## Εργαλείο AR

- `A-Frame` + `MindAR` (WebAR μέσω browser κινητού).

## Διδακτική ιδέα συν-ανάγνωσης

Η κάμερα του κινητού στοχεύει marker και πάνω του εμφανίζεται κολάζ με δύο οπτικά στρώματα. Ο μαθητής:

1. Εναλλάσσει την ορατότητα αποσπάσματος κειμένου.
2. Κρύβει/δείχνει τα layers του κολάζ για σύγκριση ερμηνειών.
3. Επιλέγει «Λογοτεχνικό Φακό» (Χώρος/Οικογένεια/Σώμα).
4. Πατά hotspot για καθοδηγούμενο σχόλιο.

## Παιδαγωγικός στόχος στο 1ο μέρος

Να φανεί στην πράξη πώς η AR στρωματοποίηση (εικόνα + κείμενο + αλληλεπίδραση) υποστηρίζει ενεργή, όχι παθητική, ανάγνωση λογοτεχνίας.
