# Metamorphosis WebAR (Εκπαιδευτικό Demo)

Διαδραστικό WebAR κολάζ για τη διδασκαλία λογοτεχνίας με βάση τη *Μεταμόρφωση* του Franz Kafka.

## Στόχος (1ο μέρος εργασίας)

Υλοποίηση παραδείγματος AR συν-ανάγνωσης κειμένου και εικόνας για χρήση στη σχολική τάξη, βασισμένου σε έτοιμο MindAR example με ουσιαστικές προσαρμογές.

## Δομή

- `index.html`: AR σκηνή + HUD διεπαφή
- `js/app.js`: διαδραστική λογική
- `styles/main.css`: οπτικό ύφος και responsive προσαρμογή
- `assets/images/`: στοιχεία κολάζ
- `docs/changes-from-example.md`: τεκμηρίωση αλλαγών πάνω στο έτοιμο παράδειγμα
- `docs/part1-design.md`: περιγραφή εκπαιδευτικού σεναρίου για το 1ο μέρος

## Τοπική εκτέλεση

```bash
cd ~/Git_projects/metamorfosis
python3 -m http.server 8080
```

Άνοιξε: `http://localhost:8080`

Για κινητό στο ίδιο δίκτυο, άνοιξε `http://<IP_ΥΠΟΛΟΓΙΣΤΗ>:8080`.

## Σημείωση

Χρησιμοποιείται προσωρινά δημόσιο demo marker (`card.mind`) για άμεσο έλεγχο λειτουργίας.
